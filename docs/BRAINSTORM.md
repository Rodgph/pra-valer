# 🧠 Social OS — Brainstorming & Futuro

Este documento é um repositório de ideias, conceitos e funcionalidades planejadas para as próximas fases do Social OS. Ele serve para guiar o desenvolvimento e manter a visão agressiva e minimalista do projeto.

---

## 🌐 1. Ecossistema Social Browser
- [ ] **Modo Noturno Forçado:** Injeção global de filtro CSS para sites que não possuem Dark Mode nativo.
- [ ] **Barra de Favoritos Brutalista:** Uma linha de ícones (favicon apenas) logo abaixo da barra de URL.
- [ ] **Ad-Block Minimalista:** Injeção de scripts para remover seletores de anúncios comuns (Google Ads, etc.) via CSS Injector automático.
- [ ] **Leitor de Modo Foco:** Extração do conteúdo principal do site para uma visualização limpa, com tipografia mono e fundo transparente.

## 🔍 2. Spotlight Search (Power User)
- [ ] **Calculadora de Kernel:** Digitar `150 * 2` e ver o resultado instantâneo no Spotlight.
- [ ] **Conversor de Unidades:** `/conv 10usd brl` para câmbio ou medidas.
- [ ] **Gerenciador de Área de Transferência:** `/clip` para ver e buscar no histórico de CTRL+C.
- [ ] **Navegação por Histórico:** Teclas ↑/↓ para recuperar os últimos 10 comandos executados.

## 📐 3. Engine de Layout & Workspaces
- [ ] **Workspaces Automáticos:** Alternar entre setups (ex: `dev`, `gaming`, `social`) baseado no horário do dia.
- [ ] **Pane Tabs:** Capacidade de ter abas dentro de um único painel fixo (empilhamento de módulos).
- [ ] **Snap Assist Brutalista:** Destaque visual agressivo ao arrastar janelas flutuantes para as bordas para "encaixá-las".

## ⚙️ 4. Módulos de Sistema (Kernel)
- [ ] **Chat IRC/Matrix:** Módulo real de comunicação com estética de terminal.
- [ ] **Terminal Nativo:** Um shell real embutido dentro de um painel do Social OS.
- [ ] **Media Controller:** Widget flutuante que detecta música tocando no navegador ou no Windows (Spotify/YouTube).
- [ ] **Visualizador de Logs:** Streaming em tempo real dos logs do Rust/Tauri com sintaxe colorida.

## 🎨 5. DNA Visual & Som
- [ ] **Feedback Sonoro:** Cliques e transições com sons de "click" mecânico ou sons de interface de computador antigo (8-bit/Lo-fi).
- [ ] **Efeito Glitch de Foco:** Uma animação rápida de interferência visual quando uma janela ganha foco.
- [ ] **Temas de Cor:** Suporte a variações (ex: Âmbar, Ciano, Vermelho Alerta) mantendo o contraste agressivo.
